Bem-vindo ao TechStore Manager

O TechStore Manager é um sistema web de gestão comercial voltado para lojas de tecnologia. O objetivo do sistema é facilitar o controle de usuários, produtos e categorias, trazendo mais organização e praticidade para o negócio.

Gestão de Usuários

Cadastro e visualização de usuários do sistema com perfis diferentes.

Controle de Produtos

Listagem de produtos com preço, estoque e vínculo com categorias.

Organização por Categorias

Separação dos produtos por categoria para facilitar a gestão do catálogo.
